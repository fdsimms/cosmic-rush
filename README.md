## Summary

Cosmic Rush is an old school-style shooter game that draws inspiration from classics like Space Invaders and Galaga and was built from the ground up using JavaScript and CSS Canvas. 


## Languages

- JavaScript
- HTML
- CSS and CSS Canvas
